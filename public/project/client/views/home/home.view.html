Welcome to LoLComps, a data-driven website built to design optimal team compositions. Combining experience, performance, and compositional stats, you can use this tool for the compeatitive advantage.

I decided to make this tool because I play a decent amount and always thought it'd be useful to have one centralized tool to help choose from the endless pool of champions instead of clicking through countless websites as the timer runs down. There is still a lot of improvement for the scoring algorthm, and a few features that are in the works, but I think there is huge growth potential.

LoLComps is intended to be used during champion select, and as such it is layed out very similar. For each player, you see this Summoner name is optional, but will make the data more accurate. For example, if one player is very good or inexperienced with a champion, it can completely change the viability of the pick.

In addition to straight up winrate, we also analyze damage composition. Ideally a balanced team composition will have the three nearly filled up. Do note that some stats, such as shields, are not measured or reported by riot or anyone so heal supports will look artificially better. Generally speaking, there is little value in getting more than full of any of the 3 bars, as itemization will nullify the advantage.
